# magtek-card-reader

Read Triple-track Magnetic Stripe Card from Magtek Swipe Card Reader

## Dependencies and setup

- Python3
- pip

`pip install -r requirements.txt`

## Usage

`sudo ./read.py` then swipe the card when asking.
